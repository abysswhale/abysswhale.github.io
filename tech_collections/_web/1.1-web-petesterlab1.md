```

- 가상 머신(Virtual Box, Virtual Player, VMware 등)
- PentesterLab 1 ISO File
- Fiddler
```

## 환경 구성
가상 머신에 PentesterLab 1 ISO 파일을 설치 후, 구동합니다.
PentesterLab OS의 IP 주소를 확인합니다.
`ifconfig'
웹 브라우저를 통해 해당 IP 주소로 접속합니다.
